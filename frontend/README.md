# Rakuten Frontend

React + TypeScript frontend with authentication.

## Setup

```bash
npm install
npm run dev
```

Visit `http://localhost:5173`

## Environment

Create `.env`:

```env
VITE_API_URL=http://localhost:8080
```

## Features

- User signup/signin/signout
- Protected routes
- JWT authentication
- Cross-tab auth sync

## Build

```bash
npm run build
```
